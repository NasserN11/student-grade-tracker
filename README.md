# Student Grade Tracker

A C++ console application for university students to calculate GPA using the Turkish grading system.

## Features

- **Turkish Grade Support**: AA, BA, BB, CB, CC, DC, DD, FD, FF, NA
- **GPA Calculation**: Weighted average based on credits
- **Case-Insensitive**: "AA", "aa", "Aa" all work
- **Error Handeling**: Invalid grades are skipped
